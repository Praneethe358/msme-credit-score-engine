# Notebooks Directory

This folder is dedicated to Jupyter Notebooks for:
1. **Exploratory Data Analysis (EDA)**: Visualizing variables, checking correlations, distributions, missing values, etc.
2. **Model Experimentation**: Testing different model architectures, hyperparameters, feature engineering methods, and logging results.

## Naming Convention
Please prefix notebooks with numbers to define the order of execution:
- `01_eda_msme_dataset.ipynb`
- `02_model_training_experiment.ipynb`
- `03_evaluation_metrics.ipynb`
